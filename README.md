# Applied Method Systems website

Static website for Applied Method Systems LLC, prepared for GitHub and Vercel.

## Included pages

- `index.html` — homepage
- `mission.html` — mission statement page
- `privacy.html` — privacy policy page
- `assets/css/styles.css` — shared styling
- `assets/js/site.js` — mobile navigation and contact form behavior
- `assets/js/hero-video-data.js` — embedded hero background video payload
- `vercel.json` — static deployment headers

## Deployment

The site is designed to deploy as a static project on Vercel through GitHub.

## Notes

- The homepage hero uses a darkened embedded background video.
- The design intentionally avoids rounded “bubble” UI treatment and keeps a sharper industrial presentation.
- Contact actions route to `adam@newworkingorder.com`.
